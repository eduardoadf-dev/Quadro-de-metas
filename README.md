# Quadro das Metas

Aplicação web para organizar e acompanhar metas pessoais em formato visual, com cards de imagem e controle de status.

## Demonstração
Link do projeto: https://eduardoadf-dev.github.io/Quadro-de-metas/

## Funcionalidades

- Adicionar metas com imagem e título
- Alternar status: **Sonho → Progresso → Conquistado**
- Remover metas
- Dados salvos no `localStorage` (persistência local)

## Tecnologias

- HTML
- CSS
- JavaScript (vanilla)

## Como usar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Não requer instalação ou dependências
